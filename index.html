"Los preferidos de nuestros lectores

Que nuestros lectores y lectoras aman

  • Clean Code

    Este es uno de los mejores libros clásicos para principiantes ya que te enseñará todos los trucos y patrones para escribir código bueno y limpio porque no todos los códigos que se ejecutan son un código limpio. La mayoría de los programadores principiantes cometen este error, solo intentan resolver el problema y, por lo tanto, olvidan estos factores para escribir un código profesional limpio y perfecto. Un código limpio tiene que ser legible, además de estar bien estructurado para que pueda reutilizarse y depurarse de manera fácil.

    Robert C. Martin

  • JavaScript: The Good Parts

    Ofrece una mirada de cerca al lenguaje de programación a los ojos de un experto. Algunos revisores no están muy seguros de para quién fue escrito el libro, pero gracias a esta lente única, aquellos que ya están familiarizados con JavaScript encontraron este recurso muy esclarecedor. Este libro es un excelente recurso para que lo analicen los expertos y, al mismo tiempo, es una excelente pieza complementaria para otros libros más adecuados para principiantes.

    Douglas Crockford

  • Head First Design Patterns

    En cualquier momento, alguien se enfrenta a los mismos problemas de diseño de software que tú. Y lo más probable es que alguien más ya haya resuelto tu problema. Esta edición de Head First Design Patterns, ahora actualizada para Java 8, te muestra los patrones de eficacia comprobada que utilizan los desarrolladores para crear software funcional, elegante, reutilizable y flexible. Al terminar este libro, podrás aprovechar las mejores prácticas de diseño y la experiencia de quienes han superado las dificultades del diseño de software y triunfado.

    Eric Freeman, Elisabeth Robson, Bert Bates, Kathy Sierra

  • Código completado: Un manual práctico de construcción de software

    ¿Te gustaría saber cómo escribir un buen código al margen de la arquitectura del lenguaje de programación que hayas elegido? Después, podrías leer Code Complete: A Practical Handbook of Software Construction ("Código completo: un práctico manual sobre creación de software"). Este aborda de manera exhaustiva todo lo relacionado con la buena estructura de código.

    Steve McConnell
